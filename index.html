Text Elements

Commonly used text elements include bold, strong, italic, emphasized, marked, small, deleted and inserted text.

  • Bold text: used for presentation purposes to draw attention
  • Important text: similar to bold more commonly used than the bold tag
  • Italic text: also for presentation purposes to indicate a mood or different quality
  • Emphasized textspecifies the stress emphasis of its contents
  • Marked text: text which is marked or highlighted for reference or notation
  • Small Text: smaller text like copyright and other side-comments but it is usually done with CSS instead.
  • Deleted text: text that has been deleted from a document usually displayed with a line through deleted text
  • Inserted text: defines a text that has been inserted into a document
  • Cite: defines the title of a creative work
  • Code: defines computer code displayed in the browser's default monospace font.
  • Var: defines a variable in programming or in a mathematical expression usually displayed in italic
